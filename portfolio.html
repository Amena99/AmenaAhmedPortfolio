API & Database Projects

These interactive projects were built with JavaScript, utilizing Bootstrap for front-end design.

Travel Attractions GIF Finder

An app that retrieves GIFs from an API based on user's choice of travel attraction. The application is built in javascript and utilizes jquery to make ajax calls. Each GIF is dynamically added to the page; it is first inserted as a static image that animates when clicked.

"Hungry" - Restaurant Locator

Built for those times when you can't decide where to eat, "Hungry" is a fully functioning app that captures the user's location and finds a restaurant that caters to the user's cuisine of choice within a 2/3 mile radius. The application utilizes the geolocation property in javascript and the Zomato Restaurants API.

Live Updating Train Schedule

Connected to a Firebase database, this application stores train schedule data and automatically loads live times for each train's arrival (based on user's browser time). Through the form, the user can add train data that will be updated in the database.
